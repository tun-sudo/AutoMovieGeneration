1.   环境安装

     环境依赖在pyproject.toml里，请大家自行安装

     uv管理的可以直接 uv sync



2.   将剧本放入一个txt文件中，然后在main.py里面修改script_path 和 working_dir

     working_dir会保存输出的文本、图像、视频结果

     视频风格可以通过style进行描述



3.   运行main.py

     生成一张图片大概需要2~3分钟

     生成一个镜头大概需要2~3分钟



4.   example_inputs文件夹包含一些样例
